Markdown

Role: Senior Full-stack Developer and UI/UX Designer.
Task: Create a collaborative, high-performance Task Planner web application.

Core Concept:
A minimalist, aesthetic "Team Flow" planner. No feature bloat. Only high-value collaborative tools.

Technical Stack:
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- UI Components: Shadcn/UI (Dark Theme by default)
- Animations: Framer Motion
- Backend/Real-time: Supabase (Auth, PostgreSQL, Real-time subscriptions)

Key Features:
1. Shared Workspace: Tasks are visible to all authenticated group members in real-time.
2. Task Management: Create, edit, delete tasks. Each task has:
   - Title & Description
   - Priority (Low, Medium, High, Urgent) with distinct color coding.
   - Deadline (Date picker with "time remaining" countdown).
   - Assignee (User avatars).
3. Collaboration: 
   - Comment system inside each task with timestamps.
   - Real-time updates: if User A changes a task, User B sees it instantly without refreshing.
4. UI/UX Requirements:
   - Modern Dark Theme (Deep grays/blacks, accent colors like Electric Blue or Violet).
   - Fully Responsive: Must work perfectly on mobile and desktop.
   - Animations: Smooth layout transitions (Framer Motion), hover effects on task cards, and micro-interactions for button clicks.
   - Clean Dashboard: Sidebar for navigation, main area for the task board (List view or Kanban-style).

Specific UI Instructions:
- Use a glassmorphism effect for modals and cards.
- Implement a clear visual hierarchy for priorities.
- Ensure the interface feels "snappy" and lightweight.

Deliverables:
- Clean, modularized code structure.
- Responsive layout with a mobile-first approach.
- Integration logic for real-time database updates.
